Step1

安装 Vscode

Step2

在 GitHub 上拉取我的 Code 文件夹：
安装 git，在 cmd 中输入 git clone https://github.com/... (项目地址)。

Step3

在 Vscode 中打开 Code 文件夹，并安装插件。

Step4

下载 MSYS2 。
并按需求安装(在紫色 msys2 中完成操作)：

clang：
pacman -S mingw-w64-x86_64-clang
pacman -S mingw-w64-clang-x86_64-toolchain
pacman -S mingw-w64-x86_64-clang-tools-extra

gcc:
pacman -S mingw-w64-x86_64-gcc
pacman -S mingw-w64-x86_64-toolchain

更新：
pacman -Syu
pacman -Su

找到 msys 的文件夹位置，将 D:\…..\MSYS2\mingw64\bin 添加进 path 中。

Step5

编译 tools 文件夹中的 check（可能缺少 testlib.h），并把 exe 文件放入 windows 文件夹。
将 windows 文件夹加入 path 系统路径。
将 keybindings.json 中的东西复制进键盘快捷方式的源码中。
下载编程字体 FiraCode，使用 git 提取并安装即可。

常见问题

Commit 里经常会不慎合并，解决方法是回退至合并前的部分，并输入指令 git push -f，重新 commit。
